# Support

Use GitHub issues for reproducible bugs, compatibility reports, and focused
feature requests.

Before opening an issue:

- Check the latest release notes in `CHANGELOG.md`.
- Check the troubleshooting section in `README.md`.
- Confirm the charger is visible to Home Assistant as an `ACP#...` Bluetooth
  device.
- Confirm any ESPHome Bluetooth proxy supports active connections and is close
  enough to the charger.
- Remove charger PINs, private BLE addresses, private logs, Wi-Fi details, and
  private Home Assistant configuration from anything you share.

For questions about Home Assistant Bluetooth proxies, ESPHome setup, or general
Home Assistant automation design, the Home Assistant community forum may be a
better place to start.

For security concerns, follow `SECURITY.md` and do not post exploit details in a
public issue.

For electrical safety, wiring, breaker sizing, charger installation, or local
code questions, consult a qualified professional. This project cannot validate
or certify an electrical installation.
