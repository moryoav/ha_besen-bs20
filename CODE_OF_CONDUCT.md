# Code of Conduct

## Our Pledge

We want Besen BS20 for Home Assistant to be a welcoming project for people
with different backgrounds, skill levels, electrical setups, and Home Assistant
installations.

Contributors, maintainers, and users are expected to participate respectfully
and constructively in issues, pull requests, discussions, and any other project
spaces.

## Expected Behavior

Examples of behavior that helps this project:

- Be kind, patient, and constructive.
- Assume good intent when possible.
- Give feedback that is specific and useful.
- Respect different experience levels.
- Keep discussions focused on the project and the problem being solved.
- Be careful when discussing chargers, electrical installations, BLE addresses,
  PINs, logs, diagnostics, or private Home Assistant configuration.

## Unacceptable Behavior

Examples of behavior that is not acceptable:

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Repeated disruption or bad-faith arguments.
- Publishing private information without permission.
- Sharing secrets, PINs, private logs, private Home Assistant configuration, or
  private device identifiers from another person.
- Encouraging unsafe electrical work or unsafe charger use.

## Enforcement

Maintainers may remove comments, close issues or pull requests, limit
participation, or take other reasonable action when behavior harms the project
or its community.

If you see behavior that violates this Code of Conduct, please report it to the
project maintainer. If the issue involves a security vulnerability, follow
`SECURITY.md` instead of posting exploit details publicly.

## Scope

This Code of Conduct applies in all project spaces, including GitHub issues,
pull requests, discussions, and any community spaces that represent the project.
